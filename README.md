# Pinpoint

Copy file path with line numbers to clipboard.

## Output format

- Cursor on line 5: `src/index.ts#L5`
- Selection from line 5 to 12: `src/index.ts#L5-L12`

## Usage

- **Keybinding**: `Cmd+Shift+L` (mac) / `Ctrl+Shift+L` (win/linux)
- **Context menu**: Right-click in editor â†’ "Pinpoint: Copy File Reference"
- **Command palette**: `Pinpoint: Copy File Reference`
